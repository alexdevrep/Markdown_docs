
# Saltos de línea 

Para crear un salto de línea hay que dejar dos espacios en blanco.

## Buenas prácticas con los saltos de línea

* Es algo difícil ver esta característa del lenguaje **por ello es mejor usar la etiqueta HTML `<br>`**

* Tambien podemos utilizar una barra inveritda `\` pero puede ser que haya problemas de compatibilidad con algún editor de texto.
  
---

Aquí te dejo un ejemplo de salto de línea:

```
Esto es una línea <br> esto es la otra línea
```
La vista que tendremos si copiamos y pegamos el código será:

Esto es una línea <br> esto es la otra línea.
