
# Listas de definición

>Atención: Este archivo pertenece a la sintaxis avanzada del lenguaje por lo que 
>no todas las variantes de Markdown ni los procesadores soportan esta sintaxis para más información clica [aquí](https://github.com/alexdevrep/Markdown_docs/blob/main/Disponibilidad/Disponibilidad.md)

Para crear una lista de definiciones escrbimos el término en una línea y en la siguiente línea escrbimos dos puntos `:` seguido de un espacio en blanco y la definición de dicho término.

Aquí te muestro un ejemplo de como crear una lista de definiciones:

```
primer término
: definición del primer término

segundo término
: definición del segundo término
```

La salida que se nos muestra por pantalla es la siguiente:

primer término
: definición del primer término

segundo término
: definición del segundo término
